# NOTICE

> MT7986/MT7975 系列与常见射频组合相关的固件与 EEPROM 文件，说明其作用与对系统的关键影响，便于刷写与调试时参考。

| 文件名 | 类型 | 作用说明 | 关键影响 |
|---|---|---|---|
| `mt7986_rom_patch.bin` | ROM Patch | 修补 MT7986 无线子系统内部固化 ROM 代码缺陷 | 无线初始化稳定性 |
| `mt7986_rom_patch_mt7975.bin` | ROM Patch（特定射频组合） | 针对 MT7986 + MT7975 射频组合的专用 ROM 补丁 | 特定硬件兼容性 |
| `mt7986_wm.bin` | WM 管理固件 | Wireless MCU 管理固件，负责射频初始化、扫描、关联、功率控制等控制面逻辑 | Wi‑Fi 是否能正常工作 |
| `mt7986_wm_mt7975.bin` | WM 管理固件（特定射频组合） | 针对 MT7975 组合的无线控制固件，含 6GHz/功率表/校准流程支持 | 6GHz 功能、校准流程 |
| `mt7986_wa.bin` | WA 加速固件 | Wireless Accelerator，加速数据面卸载与队列调度 | 吞吐性能与 CPU 占用 |
| `mt7986_wo_0.bin` | WO 卸载引擎固件实例 0 | 数据转发与卸载处理实例 0 | 高负载性能 |
| `mt7986_wo_1.bin` | WO 卸载引擎固件实例 1 | 数据转发与卸载处理实例 1（第二处理域/核） | 双实例卸载能力 |
| `mt7986_eeprom_mt7975.bin` | EEPROM 数据（单 PHY） | 单射频板级射频校准参数（功率表、信道校准、FEM 配置等） | 发射功率与射频匹配 |
| `mt7986_eeprom_mt7975_dual.bin` | EEPROM 数据（双 PHY） | 双射频/双 PHY 板级校准参数 | 双频并行能力与功率准确性 |

## 启动加载逻辑（理论顺序）

1. 加载 ROM Patch
2. 加载 WM 管理固件
3. 加载 WA 加速固件
4. 加载 WO 卸载引擎固件
5. 读取 EEPROM 校准数据

> 说明：实际加载顺序以固件加载器/bootloader 为准，此处为概念性的逻辑顺序，便于理解各组件职责。

## 类型区别

- 固件（`rom_patch` / `wm` / `wa` / `wo`）：可执行代码，运行于 SoC 或 MCU，负责控制面或数据面逻辑。
- EEPROM：板级硬件校准与射频参数数据（非可执行），影响射频性能与功率校准。

## 注意事项

- 刷写顺序：建议先应用 ROM Patch，再加载对应的 WM/WA/WO 固件，最后写入或校验 EEPROM 数据。
- 射频组合：选择带 `mt7975` 的专用文件以匹配具体射频模块，错误的组合可能导致射频不可用或参数不匹配。
- 备份：在覆盖 EEPROM 前务必备份原始数据，避免丢失板级校准导致设备性能下降。
- 兼容性：部分固件只针对特定硬件版本或 PCB 设计，务必参考设备型号与供应商说明。
- 调试：如果出现初始化失败或性能异常，检查加载日志、校验固件版本与 EEPROM 校验和。
